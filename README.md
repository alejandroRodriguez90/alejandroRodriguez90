## 💼 Sobre mí

**Desarrollador Frontend** apasionado por crear aplicaciones web modernas, rápidas y centradas en el usuario. 

Combino un frontend sólido con conocimientos de backend para entregar soluciones completas y escalables.

- ⚛️ **Frontend**: React, JavaScript/TypeScript, HTML5, CSS3, Bootstrap y Tailwind (recomendado agregar si lo usas).
- 🐍 **Backend**: Python + Django / Django REST Framework.
- 🔌 Consumo de APIs REST, autenticación y testing con Postman.
- 💾 Bases de datos: MySQL, PostgreSQL y MongoDB.
- 🌱 Siempre aprendiendo y aplicando las mejores prácticas del desarrollo web.
